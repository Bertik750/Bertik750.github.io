Managebac Grade Calculator and More + -> Privacy Policy

This app only uses Google Analytics to gather any data. They are only used for devloper purposes and are never shared. The only permissions used by the extension in the web browser Google Chrome is "tabs" and "storage" and the only websites that allow this extension to be executed is https://www.managebac.com and https://www.managebac.cn.
The data that is collected by Google Analytics:

  • Installs and weekly users
Nothing more is collected. All data remain completely anonymous and only used for development purposes such as checking managebac functionalities in different countries. No cookies are used and all data related to assigements are stored localy and encrypted by Google Chrome storage api.
ABSOLUTELY NO grade data or personal information are stored anywhere.
All data can be deleted upon request.
This project is Open-Source and comes with no gurantee.
